# Home server

My home server services setup.


## Services
| **Service** | **Description** |
|:-------:|:------------|
| Traefik | Reverse proxy |
| Pi-hole | DNS server |
| fail2ban | Intrusion detection |
| Local CA | Self-signed certificate for HTTPS on LAN |
| Vaultwarden | Password manager |
| WireGuard | VPN |


## Installation

Clone the repository and run:
```
./install.sh
```
